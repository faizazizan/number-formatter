# Phone Number Formatter

A modern, beautiful web application to format Malaysian phone numbers.

## Features

- **Format Numbers**: Automatically formats numbers to start with `60` (e.g., `0123456789` -> `60123456789`).
- **WhatsApp Links**: Converts numbers to `wa.me` links.
- **CSV Support**: Upload and download CSV files.
- **Copy All**: One-click copy for all formatted numbers.
- **Clean UI**: Glassmorphism design with smooth animations.

## Usage

1. Enter phone numbers in the text area (separated by newlines, commas, or spaces).
2. Click **Format Numbers**.
3. Copy the results or download as CSV.
